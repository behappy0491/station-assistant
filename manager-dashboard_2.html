⛽ Manager Dashboard Setup

Connect your Firebase project to sync data across all 15 stations in real-time.

1Create a free Firebase project

Go to console.firebase.google.com → Create project → Enable Firestore Database (start in test mode) → Go to Project Settings → Add a Web App → copy the config values below.

2Set Firestore rules (in Firebase Console → Firestore → Rules)

Paste this rule so your apps can read/write:
rules_version='2'; service cloud.firestore { match /databases/{db}/documents { match /{doc=**} { allow read, write: if true; } } }
